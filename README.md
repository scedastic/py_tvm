# Time Value of Money using python


## API
- Uses FastAPI, Pydantic and uvicorn
1. Install FastAPI
    - `pip install fastapi`
    - This is the framework we are using to build the API.
2. Install Pydantic
    -  `pip install pydantic`
3. Install uvicorn
    - `pip install uvicorn`
    - This is the API server that we will use.
